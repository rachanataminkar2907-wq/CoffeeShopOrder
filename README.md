Coffee Order Builder

A responsive web application that allows users to customize their coffee order by selecting different drinks, sizes, milk options, and extras. The application dynamically updates the order summary and calculates the total price in real time.

✨ Features
Choose from multiple drinks (Espresso, Latte, Cappuccino, Matcha, Chai)
Select coffee size (Small, Medium, Large)
Customize milk options (Regular, Oat, Almond, Soy, No Milk)
Add extra toppings or ingredients
Real-time price calculation
Interactive order summary panel
Modern dark-themed user interface

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
